### Favorite Distro ? Why ?
I don't have a "single favorite distro". I like a bunch of them. <br>
And i choose one of them depending on what *i need to achieve*:
- If i need a **bullet-proof distro**, that **worked yesterday**, **works today** and **will work tomorrow**, and **if i'm willing to make some sacrifices on accessibility**: i go with [NixOS](https://nixos.org/).
- If i need a **reliable** yet **simple** system: Good old [Debian](https://www.debian.org/).
- If i need to go fast - like really fast, with my machine (e.g: Gaming): [ArchBTW](https://archlinux.org/).
- Finally (and perhaps the most complex choice), if i need to work with an embeded system or a SBC, depending on the situation:
	- [VoidLinux](https://voidlinux.org/) because it just works, with non-specific hardware or systems.
	- [Alpine](https://www.alpinelinux.org/) for the most lightweight distro
	- Even [FreeBSD](https://www.freebsd.org/) for firewalls and network related tasks.
